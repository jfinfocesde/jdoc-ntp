# Actividad 5: Implementa un CRUD con PyMongo (Trabajo Grupal)

## **Objetivo:**

El objetivo de esta actividad es implementar operaciones CRUD (Create, Read, Update, Delete) en una base de datos MongoDB utilizando la librería PyMongo en la plataforma Google Colab.

## **Descripción:**

1.  Crea una base de datos MongoDB con el nombre "dbpi".
2.  Crear  colecciones en la base de datos "dbpi" basadas en el proyecto integrador".
3.  Implementar todas las operaciones CRUD    
4.  Cerrar la conexión a la base de datos MongoDB.
