# Actividad 6: Creación y Filtrado de DataFrame en Pandas

## Descripción

En esta actividad, crearás un **DataFrame** utilizando la biblioteca **pandas** de Python. El DataFrame tendrá al menos **8 columnas** con datos de tu elección. Luego, aplicarás **10 filtros de comparación** para explorar y analizar los datos.

## Instrucciones

1.  **Creación del DataFrame**:
    -   Crea un DataFrame con al menos **8 columnas**. Puedes elegir cualquier tema o conjunto de datos que te interese (por ejemplo, datos de ventas, información de clientes, registros meteorológicos, etc.).
    -   Asegúrate de que cada columna tenga un nombre significativo.
2.  **Aplicación de Filtros**:
    -   Utiliza al menos **10 filtros de comparación** para explorar los datos en el DataFrame. Puedes filtrar por valores numéricos, fechas, cadenas, etc.
    -   Ejemplos de filtros:
        -   Filtrar registros donde el valor de una columna es mayor que X.
        -   Filtrar registros donde una columna contiene una cadena específica.
        -   Filtrar registros basados en una condición lógica (por ejemplo, combinando múltiples columnas).
3.  **Documentación**:
    -   Comenta tu código para explicar qué estás haciendo en cada paso.
    -   Describe brevemente el propósito de cada filtro aplicado.
